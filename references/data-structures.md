## 考纲知识框架

### 第一章 线性表（考频：高）

**考纲要求：**
- 线性表的定义和基本操作
- 顺序存储和链式存储
- 线性表的应用

**高频考点：**
1. **顺序表 vs 链表对比**（选择题常考）
   - 存取方式：随机存取 vs 顺序存取
   - 插入删除：O(n) vs O(1)（已知节点位置时）
   - 存储密度：顺序表更高
   - 适用场景：频繁查找用顺序表，频繁增删用链表

2. **单链表操作**（大题高频）
   - 头插法和尾插法建表
   - 链表逆置（三指针法）
   - 查找倒数第k个节点（快慢指针）
   - 判断链表是否有环（Floyd判圈）
   - 两个有序链表合并

3. **双向链表和循环链表**
   - 双向链表插入删除的指针修改顺序
   - 循环链表的判空条件

**易错点：**
- 头结点和首元结点的区别
- 链表操作中指针修改顺序错误导致断链
- 顺序表的移动次数计算：插入平均 n/2，删除平均 (n-1)/2

### 第二章 栈、队列和数组（考频：高）

**高频考点：**
1. **栈的应用**
   - 括号匹配
   - 表达式求值（中缀转后缀、后缀求值）
   - 递归的非递归实现
   - 出栈序列合法性判断（Catalan数）

2. **队列的应用**
   - 循环队列的判空判满（牺牲一个空间法 vs 计数器法 vs 标志位法）
   - 层序遍历
   - 用两个栈实现队列

3. **特殊矩阵压缩存储**
   - 对称矩阵：只存下三角，k = i(i-1)/2 + j（i≥j）
   - 三角矩阵、三对角矩阵的地址计算
   - 稀疏矩阵的三元组表和十字链表

**易错点：**
- 循环队列 `(rear + 1) % MaxSize == front` 是队满而非队空
- 共享栈的栈满条件：`top1 + 1 == top2`
- 矩阵压缩存储的下标起始（从0还是从1）

---

### 第三章 树与二叉树（考频：极高）

**高频考点：**
1. **二叉树性质**（选择题必考）
   - n₀ = n₂ + 1（叶子节点 = 度为2的节点 + 1）
   - 第i层最多 2^(i-1) 个节点
   - 深度为k的二叉树最多 2^k - 1 个节点
   - 完全二叉树：n个节点的深度为 ⌊log₂n⌋ + 1

2. **二叉树遍历**（大题高频）
   - 先序、中序、后序的递归和非递归实现
   - 已知先序+中序（或后序+中序）重建二叉树
   - 层序遍历（队列实现）
   - 线索二叉树（中序线索化最常考）

3. **二叉排序树（BST）**
   - 查找、插入、删除操作
   - 平均查找长度ASL计算
   - 删除节点的三种情况

4. **平衡二叉树（AVL）**
   - LL、RR、LR、RL四种旋转
   - 最小不平衡子树的判定
   - 含n个节点的AVL树最大深度

5. **哈夫曼树和哈夫曼编码**
   - 构造过程（每次选两个最小的）
   - WPL计算
   - 前缀编码性质

6. **B树和B+树**（选择题常考）
   - B树的插入分裂和删除合并
   - B树和B+树的区别（叶子节点链、关键字存储位置）
   - m阶B树节点关键字数范围：⌈m/2⌉-1 到 m-1

**易错点：**
- 先序+后序不能唯一确定二叉树（除非是满二叉树）
- 线索二叉树找前驱后继时，要区分有线索和无线索的情况
- AVL删除后可能需要沿路径向上多次调整

### 第四章 图（考频：极高）

**高频考点：**
1. **图的存储**
   - 邻接矩阵：适合稠密图，空间O(n²)
   - 邻接表：适合稀疏图，空间O(n+e)
   - 邻接矩阵的度计算 vs 邻接表的度计算

2. **图的遍历**（大题考点）
   - DFS（类比树的先序）和BFS（类比树的层序）
   - DFS序列和BFS序列的唯一性（取决于存储结构和起点）
   - 遍历与连通性的关系

3. **最小生成树**
   - Prim算法：从顶点出发，适合稠密图，O(n²)
   - Kruskal算法：从边出发，适合稀疏图，O(elog₂e)
   - 最小生成树不唯一的条件

4. **最短路径**
   - Dijkstra：单源最短路，不能处理负权边，O(n²)
   - Floyd：每对顶点，能处理负权边（不能有负权回路），O(n³)
   - BFS求无权图最短路

5. **拓扑排序和关键路径**（大题高频）
   - 拓扑排序：每次选入度为0的顶点
   - AOE网关键路径：最早开始时间ve和最晚开始时间vl
   - 关键活动 = 最早开始 == 最晚开始的活动

6. **有向无环图（DAG）** 表达式优化

**易错点：**
- Dijkstra不适用于负权边（贪心前提被破坏）
- 拓扑排序结果不唯一
- 关键路径可能不唯一，缩短非关键路径不能缩短工期

---

### 第五章 查找（考频：高）

**高频考点：**
1. **顺序查找和折半查找**
   - ASL计算（成功和不成功）
   - 折半查找判定树的构造
   - 折半查找的前提：有序顺序表

2. **散列表**（大题考点）
   - 散列函数：除留余数法最常考
   - 冲突处理：线性探测、平方探测、链地址法
   - 装填因子与查找效率的关系
   - ASL的计算（成功和不成功分开算）

**易错点：**
- 折半查找不成功的ASL计算要用判定树的外部节点
- 散列表删除只能做标记，不能直接删（线性探测法）
- 平方探测法可能探测不到所有位置

### 第六章 排序（考频：高）

**高频考点：**

| 算法 | 最好 | 平均 | 最坏 | 空间 | 稳定性 |
|------|------|------|------|------|--------|
| 直接插入 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
| 冒泡 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
| 选择 | O(n²) | O(n²) | O(n²) | O(1) | 不稳定 |
| 希尔 | - | - | - | O(1) | 不稳定 |
| 快速 | O(nlogn) | O(nlogn) | O(n²) | O(logn) | 不稳定 |
| 归并 | O(nlogn) | O(nlogn) | O(nlogn) | O(n) | 稳定 |
| 堆排序 | O(nlogn) | O(nlogn) | O(nlogn) | O(1) | 不稳定 |
| 基数 | O(d(n+r)) | O(d(n+r)) | O(d(n+r)) | O(r) | 稳定 |

1. **快速排序**（大题高频）
   - 划分过程的手动模拟
   - 最坏情况：已排序序列，退化为O(n²)
   - 优化：三数取中选pivot

2. **堆排序**
   - 建堆过程（从最后一个非叶节点开始下沉调整）
   - 堆的插入（上浮）和删除堆顶（下沉）
   - 大根堆 vs 小根堆

3. **归并排序**
   - 二路归并的过程
   - 空间复杂度O(n)的来源
   - 外部排序中的应用

4. **排序算法选择**（选择题常考）
   - 基本有序用直接插入
   - 大规模数据用快排（平均最快）
   - 需要稳定性用归并
   - 空间受限用堆排序

**易错点：**
- 选择排序不稳定（反例：{5, 5, 2}）
- 快排的空间复杂度是递归栈深度O(logn)，最坏O(n)
- 堆排序建堆时间O(n)，不是O(nlogn)
- 基数排序是非比较排序，不受O(nlogn)下界限制

## 大题解题模板

### 算法设计题（15-20分）

答题格式：
```
算法思路：（2-3句话概括核心思想）
时间复杂度：O(...)
空间复杂度：O(...)

算法实现：
void Solution(参数) {
    // 具体代码，关键步骤加简要注释
}
```

常见出题方向：
1. 链表操作（逆置、合并、查找、删除重复）
2. 二叉树递归算法（求深度、判断对称、路径问题）
3. 图的DFS/BFS应用
4. 排序算法变形
