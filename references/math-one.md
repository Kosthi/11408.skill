## 一、高等数学（~82分，56%）

### 1. 函数、极限与连续（考频：高）

**高频考点：**
1. **极限计算**（选择题+填空题高频）
   - 七种未定式：0/0、∞/∞、0·∞、∞-∞、1^∞、0^0、∞^0
   - 洛必达法则：需验证条件（0/0或∞/∞型）
   - 等价无穷小替换（仅限乘除因子）：
     - sinx ~ x, tanx ~ x, arcsinx ~ x, arctanx ~ x
     - 1-cosx ~ x²/2, e^x-1 ~ x, ln(1+x) ~ x
     - (1+x)^α-1 ~ αx
   - 夹逼准则（数列极限常用）
   - 泰勒展开求极限

2. **连续与间断**
   - 间断点分类：第一类（可去、跳跃）、第二类（无穷、振荡）
   - 闭区间连续函数三大定理：最值、介值、零点

**易错点：**
- 等价无穷小不能在加减法中直接替换
- 洛必达法则的结果不存在不代表原极限不存在
- 1^∞型要用 e^{lim f(x)·ln g(x)} 处理

---

### 2. 一元函数微分学（考频：极高）

**高频考点：**
1. **导数与微分**
   - 导数定义的应用（特别是分段函数在分段点的可导性）
   - 隐函数求导、参数方程求导
   - 高阶导数（莱布尼茨公式、常见函数的n阶导）

2. **中值定理**（大题高频，证明题核心）
   - 罗尔定理：f(a)=f(b)，闭区间连续开区间可导 → ∃ξ使f'(ξ)=0
   - 拉格朗日中值定理：f(b)-f(a) = f'(ξ)(b-a)
   - 柯西中值定理：[f(b)-f(a)]/[g(b)-g(a)] = f'(ξ)/g'(ξ)
   - 泰勒中值定理（带拉格朗日余项）
   - 中值定理证明题的辅助函数构造策略

3. **单调性、极值与最值**
   - f'(x)判断单调性
   - 极值的充分条件（一阶/二阶）
   - 最值在闭区间端点和驻点/不可导点处取

4. **凹凸性与拐点**
   - f''(x) > 0 凹，f''(x) < 0 凸
   - 拐点：f''(x)变号处

5. **渐近线**
   - 水平渐近线：lim_{x→±∞} f(x)
   - 垂直渐近线：lim_{x→x₀} f(x) = ∞
   - 斜渐近线：a = lim f(x)/x, b = lim[f(x)-ax]

**易错点：**
- 可导必连续，连续不一定可导
- 极值点不一定是驻点（尖点也可能是极值点）
- 中值定理的ξ是开区间内的点，不包含端点

---

### 3. 一元函数积分学（考频：极高）

**高频考点：**
1. **不定积分**
   - 第一类换元（凑微分）
   - 第二类换元（三角替换、根号替换）
   - 分部积分：∫udv = uv - ∫vdu
   - 有理函数积分（部分分式分解）

2. **定积分**（大题高频）
   - 牛顿-莱布尼茨公式
   - 定积分的性质（奇偶性、周期性简化计算）
   - 变上限积分求导：F'(x) = f(φ(x))·φ'(x)
   - 定积分的应用：面积、体积、弧长

3. **反常积分**
   - 无穷区间反常积分
   - 无界函数反常积分
   - 收敛判别法（比较判别法、极限判别法）
   - 常用结论：∫₀^∞ 1/x^p dx，p>1收敛，p≤1发散

**易错点：**
- 分部积分的选取原则：反对幂三指（LIATE法则）
- 变上限积分求导时不要忘记链式法则
- 反常积分的瑕点可能在区间内部，需要分开计算

---

### 4. 多元函数微分学（考频：高）

**高频考点：**
1. **偏导数与全微分**
   - 偏导数的计算
   - 全微分存在的条件
   - 复合函数求偏导（链式法则/画树形图）
   - 隐函数求偏导

2. **极值与最值**（大题考点）
   - 无条件极值：AC-B²判别法
     - A = f_xx, B = f_xy, C = f_yy
     - AC-B² > 0 且 A > 0 极小值，A < 0 极大值
     - AC-B² < 0 鞍点
   - 条件极值：拉格朗日乘数法

3. **方向导数和梯度**
   - 方向导数 = 梯度 · 单位方向向量
   - 梯度方向是函数增长最快的方向

**易错点：**
- 偏导数存在不一定连续，连续必可微，可微必偏导存在
- 复合函数求偏导要注意中间变量的关系
- 拉格朗日乘数法别忘了约束条件本身也是方程之一

---

### 5. 多元函数积分学（考频：高）

**高频考点：**
1. **二重积分**（大题常考）
   - 直角坐标：先y后x 或 先x后y
   - 极坐标：x²+y²出现时优先考虑
   - 交换积分次序
   - 利用对称性和奇偶性简化

2. **三重积分**
   - 先一后二（投影法）
   - 先二后一（截面法）
   - 柱坐标和球坐标变换

3. **曲线积分和曲面积分**
   - 第一类（弧长/面积）和第二类（坐标）
   - 格林公式：∮(Pdx+Qdy) = ∬(∂Q/∂x - ∂P/∂y)dxdy
   - 高斯公式和斯托克斯公式
   - 路径无关的条件：∂P/∂y = ∂Q/∂x

**易错点：**
- 格林公式要求区域边界取正方向（逆时针为正）
- 极坐标变换时别忘了雅可比行列式 r
- 高斯公式要求曲面封闭，不封闭时需补面

---

### 6. 常微分方程（考频：高）

**高频考点：**
1. **一阶ODE**
   - 可分离变量：dy/dx = f(x)g(y)
   - 齐次方程：令u=y/x
   - 一阶线性：y'+P(x)y=Q(x)，公式法
   - 伯努利方程：y'+P(x)y=Q(x)y^n

2. **高阶ODE**（大题常考）
   - 二阶常系数齐次：特征方程 r²+pr+q=0
     - 两不等实根：y = C₁e^{r₁x} + C₂e^{r₂x}
     - 两相等实根：y = (C₁+C₂x)e^{rx}
     - 共轭复根α±βi：y = e^{αx}(C₁cosβx + C₂sinβx)
   - 二阶常系数非齐次：特解形式由自由项决定
     - f(x)=e^{λx}P_m(x)型
     - f(x)=e^{αx}[P_l(x)cosβx + P_n(x)sinβx]型

3. **微分方程的应用**

**易错点：**
- 特解中x的幂次取决于λ是否为特征根（以及重根次数）
- 一阶线性方程的积分因子要记准公式
- 伯努利方程先除以y^n再换元

---

### 7. 无穷级数（考频：高，数学一特色）

**高频考点：**
1. **数项级数收敛判别**
   - 正项级数：比较、极限比较、比值、根值判别法
   - 交错级数：莱布尼茨判别
   - 绝对收敛与条件收敛

2. **幂级数**（大题常考）
   - 收敛半径和收敛域（端点单独讨论）
   - 幂级数求和（逐项求导/积分后还原）
   - 函数展开为幂级数（泰勒/麦克劳林）
   - 常用展开式：e^x, sinx, cosx, ln(1+x), 1/(1-x)

3. **傅里叶级数**（数学一专有）
   - 狄利克雷收敛条件
   - 傅里叶系数计算：aₙ和bₙ
   - 收敛到函数值（连续点）或左右极限平均值（间断点）
   - 奇延拓（正弦级数）和偶延拓（余弦级数）

**易错点：**
- 收敛域端点必须逐一检验（代入级数判断）
- 幂级数逐项求导/积分后收敛域可能变化
- 傅里叶级数在间断点收敛到 [f(x⁺)+f(x⁻)]/2

## 二、线性代数（~34分，22%）

### 1. 行列式（考频：中）

**高频考点：**
- 行列式的性质和计算
- 按行/列展开（代数余子式）
- 特殊行列式：范德蒙德行列式
- 抽象行列式的计算（利用性质）

---

### 2. 矩阵（考频：极高）

**高频考点：**
1. **矩阵运算**
   - 乘法不满足交换律
   - 转置、逆、伴随矩阵
   - (AB)⁻¹ = B⁻¹A⁻¹
   - (AB)ᵀ = BᵀAᵀ

2. **初等变换和矩阵的秩**
   - 三种初等变换和对应的初等矩阵
   - 利用初等行变换求逆矩阵
   - r(AB) ≤ min(r(A), r(B))

3. **矩阵方程**
   - AX=B, XA=B, AXB=C 的求解

**易错点：**
- AB=0 不能推出 A=0 或 B=0
- |kA| = k^n|A|（n阶矩阵）
- 伴随矩阵公式：A*A = AA* = |A|E

---

### 3. 向量（考频：高）

**高频考点：**
1. **线性相关性**
   - 向量组线性相关/无关的判定
   - 向量组的秩和最大无关组
   - 线性表示的充要条件

2. **向量空间**（数学一专有）
   - 基、维数、坐标
   - 过渡矩阵

**易错点：**
- n+1个n维向量必线性相关
- 向量组等价 ⟺ 秩相等且可以互相线性表示

---

### 4. 线性方程组（考频：极高）

**高频考点：**
1. **齐次方程组 Ax=0**
   - 有非零解 ⟺ r(A) < n
   - 基础解系包含 n-r(A) 个解向量
   - 通解 = k₁ξ₁ + k₂ξ₂ + ... + k_{n-r}ξ_{n-r}

2. **非齐次方程组 Ax=b**
   - 有解 ⟺ r(A) = r(A|b)
   - 通解 = 特解 + 齐次通解

3. **方程组的综合题**（大题高频）
   - 求解、判断解的结构
   - 两个方程组公共解

**易错点：**
- 非齐次方程组两个特解之差是齐次方程组的解
- 增广矩阵的秩可能等于或大于系数矩阵的秩

---

### 5. 特征值与特征向量（考频：极高）

**高频考点：**
1. **特征值计算**
   - |A-λE|=0 求特征值
   - 特征值性质：Σλᵢ = tr(A), Πλᵢ = |A|

2. **矩阵对角化**（大题高频）
   - 对角化条件：n阶矩阵有n个线性无关的特征向量
   - P⁻¹AP = Λ
   - 实对称矩阵必可对角化，且不同特征值的特征向量正交

3. **相似矩阵**
   - 相似的必要条件：特征值相同、迹相同、行列式相同、秩相同

**易错点：**
- 特征值相同的矩阵不一定相似
- k重特征值最多有k个线性无关的特征向量
- 求正交化时别忘了施密特正交化再单位化

---

### 6. 二次型（考频：高）

**高频考点：**
1. **化标准形**
   - 正交变换法（利用特征值和特征向量）
   - 配方法

2. **正定二次型/正定矩阵**
   - 充要条件：所有特征值 > 0（等价于各阶顺序主子式 > 0）
   - 必要条件：主对角元素 > 0，行列式 > 0

**易错点：**
- 惯性定理：标准形中正项个数唯一（正惯性指数）
- 合同变换不改变正定性

### 4. 数字特征（考频：极高）

**高频考点：**
1. **期望和方差**
   - E(aX+bY) = aEX + bEY
   - D(X) = E(X²) - [E(X)]²
   - D(aX+bY) = a²DX + b²DY + 2abCov(X,Y)
   - 独立时 D(X+Y) = DX + DY

2. **协方差和相关系数**
   - Cov(X,Y) = E(XY) - EXEY
   - ρ = Cov(X,Y) / √(DX·DY)
   - 不相关 ⟺ Cov=0 ⟺ E(XY)=EXEY

3. **常见分布的期望和方差**

   | 分布 | E(X) | D(X) |
   |------|------|------|
   | B(n,p) | np | np(1-p) |
   | P(λ) | λ | λ |
   | U(a,b) | (a+b)/2 | (b-a)²/12 |
   | E(λ) | 1/λ | 1/λ² |
   | N(μ,σ²) | μ | σ² |

**易错点：**
- 不相关 ≠ 独立（仅对正态分布等价）
- 方差公式展开别忘了协方差项
- E(XY) ≠ EX·EY（除非独立或不相关）

### 5. 大数定律与中心极限定理（考频：中）

**高频考点：**
- 切比雪夫不等式：P(|X-μ| ≥ ε) ≤ σ²/ε²
- 辛钦大数定律：样本均值依概率收敛于期望
- 林德伯格-莱维中心极限定理：n个iid之和近似正态

### 6. 数理统计（考频：高，数学一重点）

**高频考点：**
1. **三大抽样分布**
   - χ²分布：n个标准正态平方和
   - t分布：标准正态/√(χ²/n)
   - F分布：(χ₁²/n₁)/(χ₂²/n₂)

2. **正态总体的抽样分布定理**（大题核心）
   - X̄ ~ N(μ, σ²/n)
   - (n-1)S²/σ² ~ χ²(n-1)
   - X̄与S²独立

3. **参数估计**（大题高频）
   - 矩估计法：用样本矩替换总体矩
   - 最大似然估计(MLE)：写出似然函数取对数求导
   - 估计量的评价标准：无偏性、有效性、一致性

4. **假设检验**
   - Z检验（σ²已知检验μ）
   - t检验（σ²未知检验μ）
   - χ²检验（检验σ²）

**易错点：**
- S² = Σ(Xᵢ-X̄)² / (n-1)，分母是n-1不是n
- MLE中别忘了检验极值点是否为最大值
- χ²分布的自由度是n-1不是n
