## 考纲知识框架

### 第一章 操作系统概述（考频：低）

**常考知识点：**
- 操作系统的四大特征：并发、共享、虚拟、异步
- 并发 vs 并行：并发是宏观同时微观交替，并行是真正同时
- 操作系统的运行机制：内核态 vs 用户态，系统调用

---

### 第二章 进程管理（考频：极高，大题必考区）

**高频考点：**

1. **进程与线程**（选择题常考）
   - 进程的五状态模型：创建、就绪、运行、阻塞、终止
   - 进程 vs 线程：进程是资源分配单位，线程是调度单位
   - 用户级线程 vs 内核级线程

2. **进程调度算法**（选择题+大题）
   - FCFS：先来先服务，非抢占
   - SJF/SPF：短作业/进程优先，可抢占(SRTN)或非抢占
   - 优先级调度：静态/动态优先级
   - 时间片轮转RR：时间片大小的影响
   - 多级反馈队列：综合了以上算法的优点
   - 计算周转时间、带权周转时间、等待时间

3. **进程同步与互斥**（大题核心，每年必考）
   - 临界区的四个条件：空闲让进、忙则等待、有限等待、让权等待
   - 信号量机制（PV操作）
     - P(S)：S-- ，若S<0则阻塞
     - V(S)：S++ ，若S≤0则唤醒一个阻塞进程
   - 互斥信号量初值=1，同步信号量初值=0或资源数量

4. **经典同步问题**（大题高频，变形考查）
   - **生产者-消费者问题**
     ```
     semaphore mutex = 1;    // 互斥访问缓冲区
     semaphore empty = n;    // 空缓冲区数量
     semaphore full = 0;     // 满缓冲区数量

     Producer:              Consumer:
       P(empty);              P(full);
       P(mutex);              P(mutex);
       放入产品;               取出产品;
       V(mutex);              V(mutex);
       V(full);               V(empty);
     ```
   - **读者-写者问题**（读优先、写优先、公平策略）
   - **哲学家进餐问题**（死锁避免策略）
   - **吸烟者问题**

5. **死锁**（选择题+大题）
   - 死锁四个必要条件：互斥、请求保持、不可剥夺、循环等待
   - 死锁预防：破坏四个条件之一
   - 死锁避免：银行家算法
     - 安全序列的判定过程
     - Need = Max - Allocation
     - Available ≥ Need_i 则可分配
   - 死锁检测：资源分配图化简

**易错点：**
- PV操作的顺序不能随意交换，同步P在互斥P之前（否则死锁）
- 信号量的值可以为负数，|负值| = 等待队列中的进程数
- 银行家算法中要检查请求是否 ≤ Need 且 ≤ Available
- 死锁的必要条件是四个同时满足，只破坏一个即可预防

### 第三章 内存管理（考频：极高）

**高频考点：**

1. **内存分配方式**
   - 连续分配：单一连续、固定分区、动态分区
   - 动态分区分配算法：首次适应、最佳适应、最坏适应、邻近适应
   - 非连续分配：分页、分段、段页式

2. **分页存储管理**（大题高频）
   - 逻辑地址 = 页号 + 页内偏移
   - 页号 = 逻辑地址 / 页大小
   - 页内偏移 = 逻辑地址 % 页大小
   - 物理地址 = 页框号 × 页大小 + 页内偏移
   - 页表大小和多级页表的设计

3. **虚拟内存**（大题高频）
   - 请求分页：缺页中断处理流程
   - 页面置换算法：
     - OPT（最优，理论参照）
     - FIFO（可能Belady异常）
     - LRU（最近最久未使用，性能好但实现代价高）
     - CLOCK/改进CLOCK
   - 工作集模型和抖动

4. **分段管理**
   - 段表结构：段号 → (段基址, 段长)
   - 地址变换：先查段表得基址，加段内偏移
   - 分段 vs 分页的区别（用户可见性、碎片类型）

**易错点：**
- FIFO算法可能出现Belady异常（分配更多页框反而缺页更多）
- LRU不会Belady异常（是栈算法）
- 多级页表减少的是页表占用的连续内存，不是访问次数（反而增加了访存次数）
- 缺页率 = 缺页次数 / 总访问次数

### 第四章 文件管理（考频：中）

**高频考点：**

1. **文件目录**
   - 单级、两级、树形、无环图目录
   - 硬链接 vs 软链接

2. **文件的物理结构**（选择题常考）
   - 连续分配、链接分配（隐式/显式）、索引分配
   - 混合索引分配（Unix inode）
     - 直接地址、一次间接、二次间接、三次间接
     - 最大文件大小计算

3. **磁盘调度算法**（选择题/大题）
   - FCFS、SSTF、SCAN（电梯算法）、C-SCAN、LOOK、C-LOOK
   - 计算磁头移动总距离和平均寻道时间

4. **磁盘物理结构**
   - 存取时间 = 寻道时间 + 旋转延迟 + 传输时间

**易错点：**
- SSTF可能导致饥饿
- SCAN到达端点后反向，C-SCAN到达端点后回到起点
- 混合索引中间接索引块存储的是地址而非数据

### 第五章 输入/输出管理（考频：中低）

**高频考点：**
- I/O控制方式：程序直接控制、中断驱动、DMA、通道
- 缓冲技术：单缓冲、双缓冲、循环缓冲、缓冲池
- SPOOLing技术（假脱机）：虚拟设备，共享打印机的实现

**易错点：**
- 单缓冲处理时间 = max(C, T) + M
- 双缓冲处理时间 = max(C+M, T)

## 大题解题模板

### PV操作题
```
1. 识别并发实体（几个进程/线程）
2. 识别需要互斥的临界资源 → 互斥信号量（初值1）
3. 识别同步关系（先后顺序/生产消费） → 同步信号量
4. 写出每个进程的伪代码
5. 验证：是否满足互斥？是否有死锁？是否有饥饿？
```

### 银行家算法题
```
1. 列出各进程的Max、Allocation、Need、Available
2. 扫描所有进程，找到 Need_i ≤ Available 的进程
3. 假设该进程完成，回收资源：Available += Allocation_i
4. 重复直到所有进程完成（安全）或无法继续（不安全）
5. 输出安全序列
```

### 页面置换题
```
1. 按访问序列逐步模拟
2. 每步记录：当前页框内容、是否缺页、替换了谁
3. 最后统计缺页次数和缺页率
```
