## 考纲知识框架

### 第一章 计算机系统概述（考频：低）

**考纲要求：**
- 计算机发展历程
- 计算机系统层次结构
- 计算机性能指标

**常考知识点：**
- 冯·诺依曼体系结构五大部件
- CPI、MIPS、主频、时钟周期的关系
  - CPI = 总时钟周期数 / 指令条数
  - MIPS = 指令条数 / (执行时间 × 10⁶) = 主频 / (CPI × 10⁶)
  - CPU执行时间 = 指令条数 × CPI × 时钟周期

### 第二章 数据的表示和运算（考频：极高）

**高频考点：**
1. **数的表示**（选择题必考）
   - 原码、反码、补码、移码的转换
   - 补码表示范围：-2^(n-1) 到 2^(n-1)-1
   - 移码 = 补码符号位取反
   - IEEE 754浮点数格式（32位和64位）

2. **浮点数IEEE 754**（大题高频）
   - 单精度：1位符号 + 8位阶码（偏移127） + 23位尾数
   - 双精度：1位符号 + 11位阶码（偏移1023） + 52位尾数
   - 规格化尾数隐含1（1.M）
   - 特殊值：阶码全0（非规格化/零）、阶码全1（无穷/NaN）

3. **定点数运算**
   - 补码加减运算和溢出判断（双符号位法、进位判断法）
   - 原码/补码乘法（Booth算法）
   - 补码除法（加减交替法）

4. **浮点数运算**（大题考点）
   - 对阶（小阶向大阶对齐）
   - 尾数运算
   - 规格化（左规和右规）
   - 舍入（0舍1入法、恒置1法）
   - 溢出判断

**易错点：**
- 补码的 -0 和 +0 表示相同，原码不同
- 补码最小负数 -2^(n-1) 没有对应的正数（不对称）
- IEEE 754 阶码用移码但偏移量是 2^(k-1)-1 而非 2^(k-1)
- 浮点数对阶是小阶向大阶看齐（尾数右移、阶码增大）

---

### 第三章 存储系统（考频：极高）

**高频考点：**
1. **主存储器**
   - 存储芯片扩展：位扩展、字扩展、字位同时扩展
   - 地址译码：全译码 vs 部分译码
   - 存储器与CPU的连接（地址线、数据线、控制线）

2. **Cache**（大题高频）
   - 映射方式：直接映射、全相联、组相联
   - 替换算法：LRU、FIFO、随机
   - 写策略：写直达(Write-Through) + 非写分配 vs 写回(Write-Back) + 写分配
   - Cache命中率和平均访问时间计算
     - t_avg = H × t_c + (1-H) × t_m（同时访问）
     - t_avg = H × t_c + (1-H) × (t_c + t_m)（先访问Cache）

3. **虚拟存储器**
   - 页式虚拟存储：页表、TLB、多级页表
   - 虚拟地址到物理地址的转换过程
   - TLB + Cache + 主存的三级访问流程
   - 缺页中断处理

**易错点：**
- 组相联的组数 = Cache总行数 / 每组行数
- LRU的计数器位数 = log₂(每组行数)
- TLB是页表的Cache，不是数据的Cache
- 写回法只在替换时才写回主存

### 第四章 指令系统（考频：中）

**高频考点：**
1. **指令格式**
   - 操作码字段和地址码字段
   - 扩展操作码设计（高位固定 → 低位扩展）
   - 指令字长 vs 存储字长 vs 机器字长

2. **寻址方式**（选择题高频）
   - 立即寻址、直接寻址、间接寻址
   - 寄存器寻址、寄存器间接寻址
   - 基址寻址（程序重定位）、变址寻址（数组访问）
   - 相对寻址（PC + 偏移，用于转移指令）
   - 有效地址EA的计算

3. **CISC vs RISC**

**易错点：**
- 间接寻址可以多级（多次访存）
- 相对寻址的偏移量基准是当前指令地址还是下一条指令地址（取决于CPU设计）
- 基址寄存器内容由OS管理，变址寄存器内容由用户程序修改

---

### 第五章 中央处理器（考频：极高）

**高频考点：**
1. **CPU的功能和基本结构**
   - 数据通路的结构：单总线、双总线、专用通路
   - 寄存器：PC、IR、MAR、MDR、ACC、PSW

2. **指令执行过程**
   - 取指周期、间址周期、执行周期、中断周期
   - 指令周期 = 若干机器周期 = 若干时钟周期

3. **数据通路分析**（大题高频）
   - 单总线结构下各指令的微操作序列
   - 分析哪些微操作可以并行

4. **控制器设计**
   - 硬布线控制器：组合逻辑电路
   - 微程序控制器：控制存储器 + 微指令
   - 微指令编码方式：直接编码、字段直接编码、字段间接编码
   - 微指令格式：水平型 vs 垂直型
   - 微程序地址形成方式

5. **指令流水线**（大题高频）
   - 流水线的基本概念和时空图
   - 流水线冲突：结构冲突、数据冲突、控制冲突
   - 数据冲突的解决：转发/旁路、暂停（stall）
   - 流水线性能指标：吞吐率、加速比、效率
   - 超标量和超流水线

**易错点：**
- 流水线周期取最长阶段的时间
- 数据冲突的RAW、WAR、WAW类型区分
- 微程序控制器中，微指令条数 ≠ 机器指令条数
- 转发不能解决Load-Use冲突（需要暂停一个周期）

---

### 第六章 总线（考频：中低）

**高频考点：**
- 总线分类：数据总线、地址总线、控制总线
- 总线仲裁：集中式（链式查询、计数器定时查询、独立请求）
- 总线定时：同步通信、异步通信（全互锁、半互锁、不互锁）
- 总线带宽计算 = 总线宽度 × 总线频率

---

### 第七章 输入/输出系统（考频：高）

**高频考点：**
1. **I/O方式**（大题考点）
   - 程序查询方式
   - 程序中断方式：中断响应过程、中断优先级、中断嵌套
   - DMA方式：周期挪用/停止CPU/交替访问，DMA传送过程

2. **中断系统**
   - 中断请求、中断判优、中断响应、中断处理、中断返回
   - 单重中断 vs 多重中断
   - 中断屏蔽字的设置

**易错点：**
- DMA传送数据不经过CPU，但DMA初始化和结束需要CPU介入
- 中断响应在每条指令执行结束时检测，DMA请求在每个总线周期末检测
- 中断处理程序是软件，中断响应是硬件

## 大题解题模板

### Cache地址映射题
```
1. 确定Cache总大小、块大小、主存大小
2. 计算块内地址位数 = log₂(块大小)
3. 根据映射方式计算：
   - 直接映射：行号 = log₂(Cache行数)，标记 = 剩余位
   - 全相联：无行号，标记 = 总位数 - 块内地址位
   - 组相联：组号 = log₂(组数)，标记 = 剩余位
4. 逐个分析访问序列的命中/缺失情况
```

### 流水线计算题
```
1. 画出时空图
2. 计算执行时间：
   - 理想情况：k + (n-1) 个时钟周期（k段流水线执行n条指令）
   - 有冲突时：加上暂停的周期数
3. 吞吐率 = n / 执行时间
4. 加速比 = 非流水线时间 / 流水线时间
```
